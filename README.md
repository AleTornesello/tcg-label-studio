TCG Label Studio is a professional web application designed to help Trading Card Game (TCG) collectors and players create, manage, and print high-quality labels for their collections. Whether you're organizing Magic: The Gathering, Pokémon, or Yu-Gi-Oh! cards, this tool provides the precision and flexibility needed for perfect labeling.

## 🚀 Features

- **Advanced Template System**: Supports standard and Japanese card sizes with single, double (foldable), and full-size options.
- **Multi-Part Labels**: Intelligent layout system that handles writable and non-writable sections for complex label designs.
- **Project Management**: Create, save, and manage multiple projects simultaneously.
- **Auto-Save**: Your work is automatically saved to local storage as you type.
- **Import/Export**: Easily backup your projects or share them with others using JSON files.
- **High-Quality Export**: Export your label pages as high-resolution PNG images in a convenient ZIP archive.
- **Print Ready**: Optimized print styles to ensure your labels come out perfectly on A4 paper.
- **Rich Text Styling**: Full control over text alignment and styling (Bold, Italic, Underline).
- **User-Centric UI**: Polished interface with auto-focusing editors and intuitive navigation.

## ⌨️ Keyboard Shortcuts

- **Arrow Keys (↑ ↓ ← →)**: Navigate between label cells in the grid.
- **Escape**:
  - **Exit Text Editing**: Press while typing in a label or project name to return focus to the grid for navigation.
  - **Deselect**: Press when a cell is selected to clear the selection and close the properties panel.
- **Auto-Focus**: Selecting a cell automatically focuses the text editor for immediate typing.

## 📖 How to Use

1. **Select a Template**: Choose the label size that matches your sleeves or storage system from the sidebar.
2. **Design Your Labels**: Click on any label in the grid to open its properties. Enter your text and apply styling.
3. **Manage Pages**: Add multiple pages to your project to organize large collections.
4. **Save & Open**: Use the "File" menu to start new projects or switch between existing ones.
5. **Export & Print**: Once your design is ready, use the "Print All" button for physical labels or "Export PNG" to save digital copies.
